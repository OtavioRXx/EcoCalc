A proposta do projeto

O EcoCalc é um projeto desenvolvido para as disciplinas de Programação Web II e Banco de Dados I, pelos alunos do 2º ano de Desenvolvimento de Sistemas. Seu objetivo é facilitar o entendimento sobre a pegada de carbono, transformando cálculos e dados ambientais em informações simples, visuais e acessíveis.

Além do cálculo de emissões, o sistema também conta com uma funcionalidade de doações voluntárias, onde o usuário pode contribuir simbolicamente para ações de redução de carbono. No entanto, por se tratar de um projeto escolar, os valores não possuem destino real e servem apenas para demonstrar como a funcionalidade funcionaria em um sistema completo.

O objetivo do EcoCalc é unir tecnologia e conscientização ambiental, oferecendo uma ferramenta educativa, funcional e representativa do aprendizado adquirido durante o curso.
